# Badges de certifications

Placez ici les images des badges de certifications au format WebP.

## Format recommandé

- **Format** : WebP (ou PNG)
- **Taille** : 384-512 px (carré)
- **Nommage** : `nom-certification.webp` (ex: `aws-saa.webp`)

## Exemples de fichiers attendus

- `aws-saa.webp` - Badge AWS Solutions Architect Associate
- `cncf-cka.webp` - Badge CKA (Certified Kubernetes Administrator)
- `gcp-pca.webp` - Badge GCP Professional Cloud Architect

Les images seront référencées dans `src/data/certifications.ts` avec le chemin `/certs/nom-du-fichier.webp`.
